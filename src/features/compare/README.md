# Feature: Compare

**Route:** `/compare`

**Purpose:** Side-by-side comparison of up to 3 countries.

## Key Features
- Countries added via "Add to Compare" button (any page)
- RadarChart for visual radar comparison
- Full metric table (budget, safety, visa, internet, etc.)
- Remove individual countries, clear all

## Dependencies
- `@data/countries`
- `@context/AppContext` — compareList, addToCompare, removeFromCompare, clearCompare

## Future Plans
